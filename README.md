# Encryption-Decryption
Web-based application to encrypt and decrypt text for learning purposes.
## Features
- **User Authentication**: Secure signup and login.
- **Encryption & Decryption**: Encrypt or decrypt text using the same key.
- **History Tracking**: Each user's encryption/decryption history is saved using **localStorage** in the browser.
- **Web Interface**: Easy-to-use web interface for interacting with the app.
## Technologies Used
- HTML, CSS, JavaScript (Frontend)
- **localStorage** for saving user data and history
